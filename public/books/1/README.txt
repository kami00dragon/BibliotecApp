CARPETA PARA EL LIBRO #1
========================

Esta carpeta debe contener los archivos del libro #1 en diferentes formatos.

FORMATOS SOPORTADOS:
- PDF (.pdf)
- EPUB (.epub)
- MOBI (.mobi)
- AZW3 (.azw3)

EJEMPLO DE NOMBRES DE ARCHIVOS:
- libro.pdf
- libro.epub
- libro.mobi
- libro.azw3

INSTRUCCIONES:
1. Coloca los archivos de este libro en esta carpeta
2. Asegúrate de que los nombres de los archivos sean descriptivos
3. El sistema detectará automáticamente qué formatos están disponibles
4. Los usuarios podrán descargar el libro en los formatos que hayas cargado

NOTA: Una vez que cargues los archivos, el libro #1 aparecerá como "Disponible" en la biblioteca.
